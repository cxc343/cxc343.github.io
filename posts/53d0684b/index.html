1. 什么是 Linux

  • Linux 是一个自由开源的 类 Unix 操作系统内核，由 Linus Torvalds 在 1991 年首次发布。

  • 它不是单一操作系统，而是 Linux 内核 + 各种 GNU 工具和软件组成的系统（通常称为 Linux 发行版，例如 Ubuntu、CentOS、Debian、Fedora）。

  • 核心特点：

    1. 开源自由：可以自由使用、修改和分发。

    2. 多用户、多任务：允许多个用户同时运行多个程序。

    3. 高度可定制：可根据需求裁剪内核和软件。

    4. 稳定性和安全性强：广泛用于服务器、云计算和嵌入式设备。

2. Linux 系统结构

Linux 系统大致可以分为四个层次：

  1. 内核（Kernel）

    • 控制硬件资源（CPU、内存、设备等）。

    • 提供系统调用接口，用户程序通过它访问硬件。

  2. Shell（命令行解释器）

    • 用户与内核交互的接口。

    • 常用 Bash、Zsh 等。

  3. 系统库（System Libraries）

    • 提供程序运行所需的标准函数（如 C 语言库 glibc）。

  4. 用户空间程序

    • 应用软件、命令工具等，例如 ls, grep, vim。

3. 文件系统基础

Linux 的文件系统呈 树状结构，以 / 为根目录：

目录功能
/根目录
/bin基本命令（二进制文件）
/sbin系统管理命令
/etc配置文件
/home用户主目录
/var可变数据，如日志
/usr用户程序和库
/tmp临时文件
/dev设备文件
/proc内存映射的虚拟文件系统，显示进程信息

4. Linux 权限体系

  • Linux 是 多用户系统，每个文件都有 所有者、用户组、权限。

  • 权限类型：

    • r：可读

    • w：可写

    • x：可执行

  • 命令：

    • 查看权限：ls -l

    • 改变权限：chmod 755 file

    • 改变所有者：chown user:group file

6. 软件管理

不同发行版软件管理工具不同：

  • Debian / Ubuntu 系列：apt

  • RedHat / CentOS 系列：yum 或 dnf

  • 通用包管理：snap 或 flatpak

7. Linux 的应用场景

  1. 服务器：Web、数据库、邮件、云计算

  2. 开发环境：软件开发、DevOps、容器（Docker、Kubernetes）

  3. 嵌入式设备：路由器、智能硬件、IoT

  4. 桌面系统：Ubuntu Desktop、Fedora Workstation

  5. 网络安全：Kali Linux 用于渗透测试
